# Contributing to PDF Toolbox

Thank you for your interest in contributing to PDF Toolbox! We welcome contributions of all kinds, from bug reports to feature requests and code improvements.

## How Can I Contribute?

### Reporting Bugs
- Use the [GitHub Issue Tracker](https://github.com/PixelPewds/pdf-toolkit/issues).
- Provide a clear and descriptive title.
- Describe the steps to reproduce the bug.
- Mention your browser and operating system.

### Suggesting Enhancements
- Open a GitHub Issue.
- Explain why the enhancement would be useful.
- Describe how it should work.

### Pull Requests
1. Fork the repository.
2. Create a new branch for your feature or fix.
3. Keep your code clean and follow the existing style.
4. Ensure your changes don't break existing functionality.
5. Provide a clear description of what the PR does.

## Development Setup

1. Install Node.js.
2. Clone the repo and run `npm install`.
3. Start the dev server with `npm run dev`.

## Code of Conduct

Be kind, respectful, and professional. We are all here to build something great together.

## Questions?

Feel free to reach out via GitHub issues or discussions.

Happy coding!
